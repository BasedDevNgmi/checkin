# Product Foundation

## Information Architecture
- `Dashboard`: quick check-in, deep journaling form, searchable timeline.
- `Check-in`: focused one-minute mood capture flow.
- `Analytics`: trends, streaks, averages, and calendar heatmap.
- `Profile`: preferences, reminders, theme, export/import backup.
- `Onboarding`: first-run profile setup for local-first mode.

## Primary User Flows
1. Quick mood capture in less than 60 seconds.
2. Deep reflection with contextual journal fields.
3. Search and filter past entries by text, date, tags, and mood.
4. Export data and restore data from backup file.

## v1 Acceptance Criteria
- Users can create mood check-ins and journal entries while offline.
- Journal entries can be edited and deleted.
- Timeline supports text, date-range, mood, and tag filtering.
- Insights show streaks, averages, top tags, and a day heatmap.
- Users can configure reminder preferences and request notification permission.
- Backup export/import validates payload shape before restore.
